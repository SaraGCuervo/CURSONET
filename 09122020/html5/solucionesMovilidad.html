Implementación De Soluciones De Movilidad Con Tecnologías Web

Objetivos

El desarrollo web certificado para la W3C para moviles se ha convertido en una alternativa al desarrollo de aplicaciones nativas para los diferentes dispositivos por su simplicidad y ahorro de costes, facilitando la creación de aplicaciones para diferentes dispositivos móviles

Prerrequisitos

Estar familiarizado con el desarrollo de aplicaciones y/o páginas web y las herramientas TIC
